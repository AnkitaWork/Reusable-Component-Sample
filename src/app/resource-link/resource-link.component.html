Useful Angular Library

Note: Here I am list out important angular package and library

Component UI Library:
1. Angular Material - https://material.angular.io/
2. Ng Bootstrap - https://ng-bootstrap.github.io/#/home
3. Prime NG - https://www.primefaces.org/primeng/
4. Nebular - https://akveo.github.io/nebular/
5. Ng-Zorro - https://ng.ant.design/docs/introduce/en
